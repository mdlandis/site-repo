Welcome to Project Generator! Myself (SlippB) and the rest of the /r/CustomHearthstone Discord have spent a very long time (since the 26th June, 2016) working on an expansion for you.

The idea was to use the handy dandy Prompt Generator developed by user Maysick to create some cards. We got so carried away that we decided to make a custom expansion. Anyway, this project literally would not have been possible without you mate. Thank you so much, honestly.

You can basically input any variables you want into the generator, and it’ll randomly generate you a few stats, or a class, or effect, or rarity or whatever you want. It’s really interesting to look at the prompts underneath the cards to see just where it started it’s life: A little bit of text, igniting an idea and that idea snowballing itself into a card.
